Type Selectors

The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document.

A selector is an HTML tag at which a style will be applied

Class Selectors

  • Definition and Usage The .class selector selects elements with a specific class attribute.
  • To select elements with a specific class, write a period (.) character, followed by the name of the class.

    • Pseudo Selectors

    The :hover selector is for selecting the elements when we move the mouse on them: Google(Hover Effect)

    To style the link for visited pages we can use :visited selector : Google(Visted)

    To style the active links we can use the :active selector: Google(Active)

    ID Selectors

    • The #id selector is used to set the style of given id.
    • The id attribute is the unique identifier in HTML document.
    • The id selector is used with # character.

    Attribute Selectors

    • Attribute Selector:
    • Attribute selector is used when we want to style multiple HTML elements having the same attribute or values
    • Attribute selector is used when we want to style multiple HTML elements having the same attribute or values
    • The attribute selectors are by default case sensitive and can be written in the square brackets [].
